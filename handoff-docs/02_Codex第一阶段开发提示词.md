# Codex 第一阶段开发提示词

请开发一个本地电脑端的「美食养生贴图生成工具 V1」。

## 一、技术建议

使用：

- Electron + React 做桌面前端
- Python 后端处理视频和图片
- SQLite 本地保存历史记录、模板、配置
- 本地运行，不接云端 API

第一阶段目标是先跑通主流程，不要做复杂云端功能。

---

## 二、工具定位

这是一个本地电脑端美食/养生贴图半自动生产工具。

核心目标：

- 上传视频/图片
- 从视频中选图/截图
- 裁剪去字幕/水印
- 提高清晰度
- 调色滤镜
- 加文字
- 拼图
- 导出图片和文案

不要做：

- 小红书/抖音爬取
- 飞书数据同步
- 云端多人协作
- AI 自动生图
- AI 判断爆款
- 复杂账号后台

---

## 三、第一阶段必须实现的页面

### 1. 首页 / 历史记录

功能：

- 新建任务
- 查看最近任务
- 继续编辑
- 删除任务
- 本地保存任务

### 2. 上传素材页

功能：

- 上传单个视频
- 上传多个视频
- 上传图片
- 拖拽导入
- 粘贴图片导入
- 素材预览

### 3. 视频选图页

功能：

- 视频播放器
- 时间轴拖动
- 当前帧截图
- 自动推荐 3 张首图候选，先用本地规则，不调用 AI API
- 步骤图候选
- 手动截图
- 上传图片替换
- 粘贴图片替换

### 4. 图片处理页

功能：

- 裁剪
- 批量裁剪
- 亮度
- 对比度
- 饱和度
- 色温
- 锐化
- 降噪
- 基础清晰增强
- 加滤镜
- 加边框
- 加背景板
- 涂抹/模糊疑似水印或字幕区域

### 5. 文字样式页

功能：

- 输入首图文字
- 输入步骤图文字
- 字体选择
- 字号
- 字体颜色
- 描边颜色
- 描边粗细
- 阴影
- 加粗
- 行距
- 字间距
- 文字位置
- 半透明背景板
- 圆角背景板
- 一键应用到全部图

### 6. 拼图/比例输出页

功能：

- 选择比例：3:4、4:5、9:16、16:9、4:3、2:1、2.35:1
- 选择拼图类型：不拼图、4拼、6拼、8拼、9拼、长图
- 选择哪几张图参与拼图
- 拖动换位置
- 调整图片间距
- 调整外边距
- 调整背景色
- 导出拼图

### 7. 导出页

功能：

- 单张图片下载
- 多张图片下载
- 拼图下载
- 文案复制
- 单张图片复制到剪贴板，如果技术可行

---

## 四、成本控制要求

第一阶段不要接 AI API。

所有 AI 功能先做成提示词模板：

- 工具内置 20 套提示词
- 用户复制提示词到豆包/即梦
- 外部生成后，用户再上传/粘贴回工具

代码结构要预留 API 接口，但默认关闭。

---

## 五、重点体验要求

1. 操作简单，不要让助理每一步都做复杂选择。
2. 图片必须越来越清晰，不能处理后变糊。
3. 图片必须能裁剪字幕和水印区域。
4. 字体、颜色、描边、背景板必须可调。
5. 批量处理必须支持应用到全部图或选中图。
6. 拼图比例不要写死，必须支持竖图贴图和横图封面比例。
7. AI 生成图必须能通过上传或粘贴导回工具。
8. 手机传电脑素材时，在界面提醒：优先用 QQ、数据线、AirDrop 或原图方式传输，不建议用微信直接传图，因为微信会压缩图片。

---

## 六、第一阶段验收标准

完成后应能做到：

1. 助理上传一个或多个视频。
2. 在视频时间轴上选首图和步骤图。
3. 对图片进行裁剪、提清晰、调色。
4. 批量应用图片处理参数。
5. 给首图和步骤图加文字。
6. 调整字体、字号、颜色、描边、背景板。
7. 选择比例和拼图布局。
8. 导出单张图和拼图。
9. 复制文案。
10. 本地保存历史任务。

请先搭建项目结构，并优先实现：上传素材、视频预览截图、图片裁剪、文字上图、单图导出这五个核心功能。
