# 美食养生贴图工具 V1 功能优先级表

## P0：第一版必须做

### 1. 本地桌面工具基础框架
- 首页/任务列表
- 新建任务
- 本地历史记录
- 本地配置保存

### 2. 素材导入
- 上传单个视频
- 上传多个视频
- 上传图片
- 拖拽导入
- 粘贴图片导入
- 素材预览
- 提醒：手机传电脑素材优先用 QQ/数据线/AirDrop/原图方式，不建议微信直接传图

### 3. 视频选图
- 视频播放器
- 时间轴拖动
- 当前帧截图
- 自动推荐 3 张首图候选，使用本地规则，不调用 AI API
- 步骤图候选
- 换一批候选
- 上传图片替换
- 粘贴图片替换

### 4. 图片裁剪与基础处理
- 手动裁剪
- 批量裁剪
- 裁掉底部字幕区
- 涂抹/模糊疑似水印或字幕区域
- 亮度/对比度/饱和度/色温
- 锐化/降噪/基础清晰增强
- 基础滤镜

### 5. 批量处理
- 全选
- 选中多张
- 应用到全部
- 应用到选中
- 批量裁剪
- 批量提清晰
- 批量调色
- 批量文字样式
- 批量边框/背景板

### 6. 文字上图与样式
- 首图文字输入
- 步骤图文字输入
- 字体选择
- 字号
- 字体颜色
- 描边颜色/粗细
- 阴影
- 加粗
- 行距/字间距
- 文字位置
- 半透明背景板
- 圆角背景板
- 边框
- 一键应用到全部图

### 7. 拼图与比例输出
- 选择比例：3:4、4:5、9:16、16:9、4:3、2:1、2.35:1
- 选择拼图类型：不拼图、4拼、6拼、8拼、9拼、长图
- 选择哪几张图参与拼图
- 拖动调整顺序和位置
- 调整间距、外边距、背景色

### 8. 导出
- 单张图片下载
- 多张图片下载
- 拼图下载
- 文案一键复制
- 单张图片复制到剪贴板，如果技术可行

### 9. 成本控制
- 第一版不接 AI API
- 所有 AI 文案/生图/高清增强先做成提示词模板
- 助理复制到豆包/即梦使用，再把生成图导回工具

---

## P1：第二阶段增强

- 手机扫码上传素材到电脑端工具
- 更多模板
- 模板导入/导出
- 更强的水印/字幕涂抹
- 复制多张图片到剪贴板，如技术稳定
- 激活码 + 设备码绑定 + 到期时间
- 更好的首图候选本地评分规则
- 更多拼图布局
- 图片处理参数保存为模板
