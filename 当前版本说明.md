# 当前版本说明

## 阶段名称
美食养生贴图工具助理试用版 V1

## 稳定 commit
8276df4

## 已通过功能

1. 第二步模板套用（保存模板 / 套用到其他图）
2. 第二步批量导出 PNG / ZIP
3. 第二步原始单图导出 / 复制正文 / 导出正文 txt
4. 第二步图片清晰度优化（锐化）
5. 第三步背景模板保存 / 复用（IndexedDB 本地存储）
6. 第三步只选已保存 / 全选 / 取消全选
7. 第三步批量导出单图成品（每张参与图各自独立导出）
8. 第三步 ZIP 单图成品
9. 第三步导出当前拼图
10. 第三步正文不上图（发布正文独立于图层）
11. 第三步背景、文字、贴图、高清导出
12. 试用码授权系统（本地 HMAC-SHA256 签名，有效期验证）
13. 管理员生成试用码工具（本地 HTML，可生成 3/7/15/30 天试用码）

## 暂停功能

1. **批量改比例**  
   原因：旧方案会永久修改 baseDataUrl，导致图片变小、出现白边，影响第三步成品质量。  
   后续如做比例功能，只能在第三步做"成品模板"方式，绝不在第二步改图。

## 已知限制

- 试用码密钥内嵌于前端代码（V1 本地试用版可接受，不适合高安全场景）
- Tag push 因远程 HTTP 403 无法推送到 GitHub（Tag 仅本地存在）
- 不支持远程实时撤销试用码（依赖到期自动失效）
