美食养生贴图工具 · 助理试用版 V1
使用说明
====================================

一、如何启动
------------
1. 找到文件夹里的「启动工具.bat」
2. 双击它
3. 会弹出一个黑色命令窗口，等它自动打开浏览器
4. 浏览器打开后就可以用了
★ 重要：黑色窗口不要关闭！关闭后工具就停了。


二、如何输入试用码
------------------
1. 第一次打开工具会出现"请输入试用码"的页面
2. 把管理员发给你的试用码粘贴进去（Ctrl+V）
3. 点「进入工具」
4. 验证通过后自动进入工具
5. 下次再开，如果试用码没过期，会自动进入，不用重新输入


三、如何导入图片
----------------
1. 进入工具后，在第一步「视频选图」页面
2. 点「添加图片」按钮（推荐用 QQ 传图，不要用微信传，微信会压缩图片）
3. 选择你的素材图片（支持多选）
4. 图片会出现在右边的「素材池」


四、第二步如何做基础图
----------------------
1. 点顶部导航「成图编辑」
2. 左边选择一张素材
3. 在右侧面板调整：比例、裁剪、文字、艺术字等
4. 做好第一张后，点右上角「保存为模板」


五、如何保存模板
----------------
1. 做好第一张图的样式后
2. 点右侧「保存为模板」按钮
3. 给模板起个名字
4. 之后可以直接套用到其他图片


六、如何套用模板
----------------
1. 在左边点选另一张素材
2. 在右侧找到「套用模板」
3. 选择你之前保存的模板名称
4. 点「套用」，样式会自动应用到这张图


七、如何保存当前成图
--------------------
1. 每张图调整满意后，点右上角「保存当前」
2. 保存后图片右上角会出现绿色标记
3. 只有「保存当前」的图才会在第三步出现


八、如何批量导出基础图
----------------------
1. 在「成图编辑」页面底部
2. 勾选想导出的图片（全选按钮可以一键全选）
3. 点「批量导出 PNG」或「批量 ZIP」
4. 文件会自动下载到你的下载文件夹


九、第三步如何只选已保存
------------------------
1. 点顶部导航「拼图导出」
2. 在底部图片队列里
3. 点「只选已保存」按钮
4. 系统自动勾选上所有已保存的成图


十、如何套背景模板
------------------
1. 在第三步右侧面板
2. 找到「⑥ 背景图设置」
3. 如果有保存过背景模板，点「套用」选择模板
4. 没有时，点「上传背景图」上传一张背景图


十一、如何批量导出单图成品
--------------------------
1. 选好参与的图片后（建议先点「只选已保存」）
2. 点队列上方的「批量导出单图成品」按钮
3. 系统会逐张下载，每张都是单独的成品图


十二、如何 ZIP 打包
-------------------
1. 选好参与的图片后
2. 点队列上方的「ZIP 单图成品」按钮
3. 会下载一个 .zip 文件，里面包含所有成品图


====================================
常见问题
====================================

Q: 打不开怎么办？
A: 确认黑色窗口是否打开了。如果浏览器没自动打开，
   手动在浏览器地址栏输入：http://127.0.0.1:7777

Q: 试用码过期怎么办？
A: 联系管理员获取新的试用码，输入后重新验证即可。

Q: 导出的图在哪里？
A: 在你电脑的「下载」文件夹里，文件名一般是
   final-001.png、final-002.png 这样。

Q: 浏览器没反应怎么办？
A: 刷新一下页面（按 F5）。如果还不行，关掉浏览器，
   重新双击「启动工具.bat」。

Q: 能不能关闭那个黑色窗口？
A: 不能！黑色窗口就是服务，关掉就停了。
   工具用完后再关。
